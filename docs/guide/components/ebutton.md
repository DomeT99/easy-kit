# EButton

The `EButton` component replaces the standard html button and encapsulates well-defined logic that can be reused throughout the app.

## Usage

```js
<script setup>
import { EButton } from "easy-kit-component";
</script>

<template>
  <EButton>Hello!</EButton>
</template>
```
